# PDF Splitter

ブラウザで動作するPDF分割ツールです。ファイルをアップロードしてページを選択し、分割してダウンロードできます。

## 特徴

- **ドラッグ＆ドロップ対応** - PDFファイルを簡単にアップロード
- **ページプレビュー** - 全ページをサムネイル表示
- **柔軟な選択方法** - クリック選択またはページ範囲指定（例: 1-3, 5, 7-10）
- **2つの分割モード**:
  - 選択したページを1つのPDFにまとめる
  - 各ページを個別のPDFファイルとして保存
- **プライバシー保護** - すべての処理はブラウザ内で完結（サーバーへのアップロードなし）

## 使い方

1. `index.html` をブラウザで開く
2. PDFファイルをドラッグ＆ドロップまたはクリックして選択
3. 分割したいページをクリックして選択（またはページ範囲を入力）
4. 分割モードを選択
5. 「分割してダウンロード」ボタンをクリック

## 動作環境

- Chrome, Edge, Firefox などの最新ブラウザ
- Windows on ARM (Snapdragon X/Surface Pro 12) でも動作

## 技術スタック

- [PDF.js](https://mozilla.github.io/pdf.js/) - PDFレンダリング
- [pdf-lib](https://pdf-lib.js.org/) - PDF操作・分割
